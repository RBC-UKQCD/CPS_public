A list of some example codes

 The QCDOC Collaboration


Below are some example codes that use the CPS library to help you understand how a programme can use the CPS library. These are hyperlinked to the CPS reference manual created by doxygen to make them easier to read.

  • t_asqtad_pion
    This program computes the correlator of the goldstone pion for Asqtad fermions.

  • s_spect
    In principle this program generates some dynamical staggered configurations and then does some hadronic correlator measurements. In practice this dies in some of the hadronic measurement code.

  • t_gauge_rot
    This program computes a random gauge transform on a unit gauge configuration. The mean trace of the link is computed.

These example codes are under the test directory in the CPS code repository.
